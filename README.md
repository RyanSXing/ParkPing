# ParkPing

Resident-first parking coordination for multifamily buildings.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Admin Access

Set `ADMIN_EMAILS` to a comma-separated list of Supabase auth user emails that can access `/admin`.

## Scripts

```bash
npm run typecheck
npm test
npm run lint
```
